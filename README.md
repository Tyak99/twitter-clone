# Twitter Clone
Twitter Clone is an application that enables connection with several people and social networking service on which users post and interact with messages known as "Tweets".

## Features
---
### Users
- Signup and Login
- Create Tweets
- Modify Tweets
- Update User Profile
- Followers and Follow People

### Admin
- Change User Status
